# go-server-template
The basic server template extend for clean-architecture with common library and tool

## Package
- Router: Gin
- Unit test: Testify
- Config: Viper
- ORM: Gorm
- Logging: Zerolog
- Cmd: Cobra
- Error trace: Eris
- api doc: swaggo/swag

## Tool
- DB:
    - Postgres
    - Sqlite
- End to End test: K6
